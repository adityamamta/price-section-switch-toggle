# Pricing Plan Section
## <a href="https://adityamamta.github.io/price-section-switch-toggle/"><img src="img/readme-btn.png" alt="Click to view live website" height="120"></a>


A modern, fully responsive pricing plan component for websites. This project includes:

- Three customizable pricing plans: Free, Growth, and Pro.
- Monthly/Yearly toggle: Users can seamlessly switch between billing options.
- Interactive hover effects: Enhanced visuals for better user experience.
- Clean and scalable design: Built with accessibility and flexibility in mind.
### Key Features
- HTML, CSS, and JavaScript for dynamic functionality.
- Material Symbols icons integrated for professional appearance.
- Responsive design: Optimized for all screen sizes.
- Custom animations: Smooth transitions for price toggling and hover effects.
